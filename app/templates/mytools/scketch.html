Visão Geral

  • Converte a imagem para tons de cinza.
  • Aplica um desfoque gaussiano na imagem em tons de cinza.
  • Cria um efeito de desenho dividindo a imagem em tons de cinza pela imagem desfocada.

Como Funciona

Passo 1: Converter a Imagem para Tons de Cinza

O primeiro passo é converter a imagem de entrada do seu espaço de cores original (geralmente BGR) para tons de cinza. Isso é feito usando a seguinte linha de código: 

gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)

cv2.cvtColor é uma função do OpenCV que converte uma imagem de um espaço de cores para outro.

  • image é a imagem de entrada.
  • cv2.COLOR_BGR2GRAY é o código de conversão de espaço de cores que converte a imagem de BGR (Azul, Verde, Vermelho) para tons de cinza.

Passo 2: Aplicar Desfoque Gaussiano

Em seguida, um desfoque gaussiano é aplicado à imagem em tons de cinza para suavizá-la. Isso ajuda a criar o efeito de desenho reduzindo ruídos e detalhes na imagem. O código para este passo é:

gray_blur = cv2.GaussianBlur(gray, (21, 21), 0)

cv2.GaussianBlur é uma função do OpenCV que aplica um desfoque gaussiano na imagem.

  • gray é a imagem de entrada em tons de cinza.
  • (21, 21) especifica o tamanho do kernel usado para desfocar. Um tamanho de kernel maior resulta em um desfoque mais significativo.
  • 0 é o desvio padrão na direção X. Se definido como 0, ele é calculado com base no tamanho do kernel.

Passo 3: Criar o Efeito de Desenho

Finalmente, o efeito de desenho é criado dividindo a imagem original em tons de cinza pela imagem desfocada. Isso destaca as bordas e cria uma aparência de desenho a lápis. A linha de código relevante é: 

image = cv2.divide(gray, gray_blur, scale=256)

cv2.divide é uma função do OpenCV que divide uma imagem por outra.

  • gray é a imagem original em tons de cinza.
  • gray_blur é a imagem em tons de cinza desfocada.
  • scale=256 é um fator de escala aplicado ao resultado da divisão para garantir que os valores dos pixels de saída permaneçam dentro da faixa válida (0 a 255).
